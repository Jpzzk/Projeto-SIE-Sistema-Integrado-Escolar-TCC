Sistema Integrado Escolar (SIE)

Organização simples e eficiente para bibliotecas escolares

Sobre o Projeto

O SIE foi criado para melhorar a organização da biblioteca escolar, facilitando o controle de empréstimos e devoluções de livros. A ideia é substituir anotações manuais por um sistema mais simples, rápido e organizado.

Funcionalidades

Registro de empréstimos
Controle de devolução
Organização de usuários
Avisos de devolução

Resumo

O sistema registra informações como nome, telefone, data de retirada e devolução dos livros. Também pretende enviar avisos para evitar atrasos e melhorar o controle da biblioteca.

Objetivo

Criar um sistema digital que facilite o controle de empréstimos e devoluções, tornando tudo mais organizado e prático.
